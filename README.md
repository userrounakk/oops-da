<h1 align="center" style="border: 0;"> Result Management System </h1>

A C++ Program to manage and show result of all the students in a school.

## Features
- Auth
- Teacher
  - View All Students
  - View Class Marks
  - View Marks of an individual student
  - Edit Marks of a student
- Student
  - View My Details
  - View My Marks
  - View Statistics
